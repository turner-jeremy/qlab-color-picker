Made by Jeremy Turner


Instructions

This is a simple interface to accurately display the colours available in QLab (including hidden colours), and copy the name to your clipboard for adding to Qlab scripts.

  1. Select the version of QLab you're using (QLab 4 or 5)
  2. Select the syntax you want copied to your clipboard:
    • Just the name of the colour
    • Surround colour name in quotation marks (just to avoid forgetting them)
    • Provides the entire script for assigning colour to selected cues.
  3. Select the colour of your choice (They are sorted by hue).
  4. Paste it into your QLab script!
